# Sample Recon Audit (Public-Safe)

This folder contains a public-safe example audit that demonstrates the report format and narrative style used in the Prox Recon Toolkit.

## Contents
| File | Description |
|------|-------------|
| `sample_recon_audit.html` | Terminal-style HTML report |
| `sample_recon_audit.pdf`  | PDF export for offline viewing |

## Highlights
- Subdomain and DNS discovery summary
- Email exposure from public sources
- Service and port fingerprinting overview
- OSINT insights from public datasets
- Recommendations to reduce exposure
